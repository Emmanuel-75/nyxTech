# Simple webpage effect

A challenge designed to test practical understanding of Colors, Shadows, and Effects in Tailwind CSS

## Features
- Semantic HTML5 structure
- Clean, readable indentation
- Switched from vanilla CSS to Tailwind CSS
- Responsiveness
- Gradients and transition effects

## Tech Stack
- HTML5
- Tailwind CSS

## Purpose
Practical application of Colors, Shadows, and Effects in Tailwind CSS to create effects and transitions to give UI depth,  vibrancy, and a polished professional look.

## How to Use
Download the file and run `npm run dev` in the terminal
or
Tap on the live link associated with this folder
